# BankDatabase
This is a project to practice object oriented programming with java and put it to use.


# Description:

This is a banking database software that will allow the user to create employee or customer accounts that can be managed by the user. The user can either log in as an employee or customer, and there are different methods that are available to each. In all this program brings ease of use to employees and customers that want to view and manage their account smoothly.

Employees:
 will be able to manage accounts with methods that will be able to change the name, change address, add, or subtract account balances, create new employee or customer accounts, view and manage account information.
 
Customers:  will be able to view or change their account information through the software.

# Installation:
The project can be run through any IDE that supports java. If the project is exported, it must be exported as a .jar file because of the classes.
 If ran through the console:          java -jar <jar-BankingDatabaset>.jar
 
# Technologies:
The Banking Database software is a Java only program. There are four classes that consist of Account.java, Control.java, Customer.java, and Employee.java . The only package that is used is java.util package.
